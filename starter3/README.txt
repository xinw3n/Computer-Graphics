Build Instructions (Athena with Makefile)
==================
$ mkdir build
$ cd build
$ cmake ..
$ make

1. Compile and Run

in build directory: 
cmake .. 
make  
./a3 (method) (param) 
Method: 
e Euler 
t trapezoid 
r rk4 

Parameters that work well with 8x8 cloth: 
t 0.001 
t 0.01 
r 0.001
r 0.01

To change the cloth size, modify W and H in clothsystem.cpp and compile
